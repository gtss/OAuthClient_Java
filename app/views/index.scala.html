TrainingPeaks OAuth Client

Java Client Example

This application contains working example code. It accesses the TrainingPeaks OAuth Server for authentication and then being redirected directly into the app. To start the OAuth process, click below.

Here are the high level steps it follows

  1. Call for a token
  2. If no token is available, the OAuth server will ask the user for their TP credentials
  3. After authenticating the credentials, a token will be handed back to this client and the callback url will be triggered
  4. Using the token, the client will prepare a resource request and send another request to the OAuth server
  5. The OAuth server will respond with the URL the client can use to access the TrainingPeaks application
